CSJMU Inventory Verification System

PHASE 2 AUTOMATION

Features:
- Department-wise inventory verification
- QR code tracking
- Google Sheet integration
- Missing asset reports
- Dashboard generation
- Audit reports

How to Run:

1. Install Python
2. Install requirements
3. Put department Excel files inside input_excels/
4. Run merge_inventory.py
5. Run generate_qr.py
6. Run generate_reports.py
7. Run dashboard.py
